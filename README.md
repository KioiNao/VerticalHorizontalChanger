# [縦書き・横書き相互変換ツール](https://kioinao.github.io/VerticalHorizontalChanger)
## 変換設定について

本ツールでは、縦書き原稿を横書きのWeb小説向け表記に整えるため、以下の変換機能を用意しています。  
各機能はチェックボックスで個別に有効／無効を切り替えられます。

---

### 濁点位置移動（『゛あ』 ⇔ 『あ゛』）

縦書き環境で入力された濁点表記を、横書き向けの表記に変換します。  
また、その逆方向の変換にも対応しています。

**例**
- `゛あ` → `あ゛`
- `あ゛` → `゛あ`

---

### 感嘆符変換（『⁉』⇔『！？』 / 『‼』⇔『！！』）

Unicodeの感嘆符・疑問符記号と、一般的な全角記号表記との相互変換を行います。

**例**
- `⁉` → `！？`
- `‼` → `！！`

---

### 会話文前後改行挿入／削除

会話文（カギカッコで囲まれた文章）を検出し、前後に改行を挿入、または削除します。  
縦書き原稿で詰めて書かれた会話文を、横書きで読みやすいレイアウトに整えます。  
`「」`のみ対応しています。 `『』`や`（）`には対応しておりません。

---

### 漢数字 → 算用数字変換

漢数字表記を算用数字に変換します。

**例**
- `レベル一` → `レベル1`
- `三・一四` → `3.14`
- `一億五千万` → `1億5000万`

文章としての自然さを重視し、  一部の慣用句・熟語表現は変換対象から除外しています。  
また、変換結果は一覧で確認でき、最終的な適用は手動で選択できます。  
読みやすさを優先するため、`万` 以上の単位は意図的に変換していません。

※ 算用数字 → 漢数字 の変換には対応していません。

---

本ツールは、機械的な一括変換ではなく、  
**縦書きで書いた小説を、横書きでも違和感なく読める形に整えるための補助ツール**として設計しています。
